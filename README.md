# Inventory Optimization Project

## 📊 Overview
This project analyzes SKU-level inventory consumption using Excel and Power BI to identify inventory concentration and cost drivers.

## 🎯 Objective
To optimize inventory management by identifying high-value SKUs and recommending a tiered inventory strategy.

## 🛠 Tools Used
- Microsoft Excel (ABC Analysis)
- Power BI (Dashboard & DAX)
- Word/PDF (Reporting)

## 📈 Key Insights
- Category A SKUs account for 79.85% of total consumption value
- Inventory turnover ratio: 4.43
- Total estimated cost: £138K
- Holding and ordering costs are major cost drivers

## 💡 Recommendations
- Tight replenishment for Category A SKUs
- Balanced strategy for Category B
- Low-frequency ordering for Category C

## 📁 Files Included
- Excel dataset
- Power BI dashboard
- Outcome report
- Professional summary

## 🚀 Outcome
Improved understanding of inventory concentration and cost optimization strategy.
